About Project

This system can be used for Driving Licence (DL) and the Certificate of Registration (RC) of vehicles using Quick Response Code (QR Code) , which will make it easy for officials to check if the licence is genuine.

Once the QR code is scanned, an official will be able to view the driver's name, residential address, validity of the licence and types of vehicles they are authorised to drive.

This concept can also be applied to other places where we have to quickly verify the document authenticity.
